# Banana Budgeting Tool

Banana budgeting tool created for Bob!

# Installation

**Clone** or **Download** the repo.

Run npm install to download the required libraries.

```
npm install
```

Run npm start & npm run dev to initiate the app.

```
npm start
npm run dev
```
